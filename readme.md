#### How to apply custom configuration

##### For vanilla `opencode`:
- Install [opencode](https://github.com/anomalyco/opencode).
- Move everything from `/opencode` folder to a global configuration path: `~/.config/opencode`.
- Replace files if needed.
- Restart `opencode`.

##### For `opencode` + `oh-my-opencode` plugin:
- Install [opencode](https://github.com/anomalyco/opencode) first and [oh-my-opencode](https://github.com/code-yeongyu/oh-my-openagent) then.
- Move everything from `/opencode+oh-my-opencode` folder to a global configuration path: `~/.config/opencode`.
- Replace files if needed.
- Restart `opencode`.
- bonus: you can ask `opencode` to make an alias (for example `omo`) that checks a version of the plugin and updates it if needed. 

Have fun!
